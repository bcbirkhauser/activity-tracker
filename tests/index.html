When you want to track Activity like this, you simply add 

data-track="category"
to the attribute you want to track.

This link will track the id because it is actually a div and not an anchor tag.

Alternatively, you can manually call the track function.

To get the data, you simply call the function on the tracker

var data = ActivityTracker.getData();
